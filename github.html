GitHub

GitHub Account Creation To use Git and GitHub, it is necessary to create a user account from this address (https://github.com/). After creating the account, you can start using GitHub.




There are three important concepts to know: repository, branch, and fork.




Repository (warehouse):





Projects hosted on GitHub. It can be considered as the main folder of the project. A separate repo is created for each project.



Branch (branching):







If the project is ready for publication for a while and something is to be tried, a new copy can be produced and branched out from the ready project by branching. This produced copy can be combined with a working copy if desired. Also, the branch is used to produce different versions from the project. Branch can be thought of as the subfolders of the repository.





Fork (forking):







To start working on a project, you can fork the project and get a copy-clone of your account. After forking the project, you can start working on it.
